### Download Go:
https://golang.org/dl/go1.15.windows-amd64.msi

### Using go modules:
```
go mod init <name_modules>
```

### Build project
```
go build
```

### Run project
```
go run main.go
```
or
```
./<name_modules> (./redis-golang-cnwnc)
```

### Using package  
https://github.com/go-redis/redis
